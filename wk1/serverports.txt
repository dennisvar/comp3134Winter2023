1. What is the definition of server port?

A server port is an endpoint on a network that is identified by unique number for a specific process, application or service.

2. What is server port range?

Server port range is the range of port numbers that can be used on a server. In the TCP/IP model port numbers range from 0 to 65535 which is the highest number that can be represented by a 16 bit unsigned integer. 0 to 1023(2^10-1) are well knwon ports, 1024 to 49151(2^14+2^15-1) are registered ports and 49151 to 65535(2^16-1) are private ports.
